# library-inventory-manager--kunal-singh-
Library Manager - Python CLI Project
A clean, modular library management system built using Python OOP.

Features
Add books with title, author, ISBN
Issue and return books
Search by title or author
View full library inventory
CLI menu interface
Logging for all operations
Project Structure
library_manager/ — Core classes
cli/ — Command-line interface
tests/ — Unit tests (optional)
requirements.txt — Dependencies
